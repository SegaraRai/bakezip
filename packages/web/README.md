# BakeZip Web

This is the web interface for BakeZip, built with [Astro](https://astro.build) and [Svelte](https://svelte.dev).

## Scripts

- `pnpm dev`: Start the development server.
- `pnpm build`: Build the project for production.
- `pnpm check`: Run type checking and Svelte checks.
- `pnpm preview`: Preview the production build locally.
- `pnpm render-social`: Render social media assets.

## Dependencies

This project depends on the `bakezip` WASM package from the workspace.
